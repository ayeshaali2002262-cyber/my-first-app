# Usage Guide - Slide to Notes Converter

## Quick Start Guide

### Step 1: Upload Your Slides

There are two ways to upload slides:

**Method A: Drag and Drop**
1. Take screenshots or export your presentation slides as images
2. Drag the image files directly into the upload area
3. Drop them when you see the blue highlight

**Method B: Click to Browse**
1. Click anywhere in the upload area
2. Select one or multiple slide images from your computer
3. Click "Open" to upload

**Supported Formats:**
- PNG (.png)
- JPEG (.jpg, .jpeg)
- Multiple files can be uploaded at once

### Step 2: Review Uploaded Files

After uploading, you'll see:
- Thumbnail preview of each slide
- File name
- Option to remove individual files
- "Clear All" button to start over

### Step 3: Convert to Notes

1. Click the **"Convert to Notes"** button
2. Wait for processing (you'll see a loading indicator)
3. Processing time depends on:
   - Number of slides
   - Image resolution
   - Amount of text on slides

### Step 4: Review Generated Notes

The notes panel will display:
- **Slide thumbnail** - Visual reference
- **Slide number** - Order maintained
- **Bullet points** - Organized, comprehensive notes
- **Bold headings** - Main topics and titles
- **Sub-bullets** - Supporting details
- **Raw text** - Expandable section showing original OCR output

### Step 5: Export Your Notes

Choose your preferred export format:

**üìã Copy Button**
- Copies all notes to clipboard
- Paste into any application
- Great for quick sharing

**üìÑ TXT Button**

**üìù MD Button** (Recommended)

- Creates hierarchical structure with sub-bullets
- Preserves all information from slides

## Troubleshooting

### No Text Detected
**Problem:** OCR returns "No text detected"
**Solutions:**
- Check image quality and resolution
- Ensure text is clearly visible
- Try adjusting image contrast
- Re-export slides at higher quality

### Incomplete Text Extraction
**Problem:** Some text is missing from notes
**Solutions:**
- Use higher resolution images
- Ensure text isn't too small
- Check for text overlapping with graphics
- Verify image isn't corrupted

### Slow Processing
**Problem:** Processing takes too long
**Solutions:**
- Reduce number of slides per batch
- Use smaller image file sizes
- Close other browser tabs
- Try a different browser (Chrome recommended)

### Export Not Working
**Problem:** Can't download or copy notes
**Solutions:**
- Check browser download permissions
- Disable pop-up blockers
- Try different export format
- Clear browser cache

## Best Practices

1. **Organize Before Upload**
   - Name files sequentially (slide1.png, slide2.png)
   - Keep slides in order
   - Remove duplicate slides

2. **Review Before Export**
   - Check generated notes for accuracy
   - Expand raw text if needed
   - Verify all slides processed correctly

3. **Choose Right Export Format**
   - Markdown for structured notes
   - Text for simple sharing
   - Copy for quick paste

4. **Maintain Quality**
   - Always use original slide files
   - Avoid re-compressed images
   - Keep aspect ratio intact
